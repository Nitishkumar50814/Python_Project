#  Fake News Headline Generator (Python)

This is a beginner-friendly **Python project** that generates random **fake news headlines** in two modes:
-  Funny Mode
-  Serious Mode

The project uses simple Python concepts like lists, loops, conditionals, user input, and the `random` module.

---

##  Features

- Two headline modes: **Funny** and **Serious**
- Random selection of subjects, actions, and places
- Continuous headline generation using a loop
- Easy-to-read and beginner-friendly code
- User-controlled execution (yes/no)
